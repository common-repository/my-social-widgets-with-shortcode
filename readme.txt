== Installation ==
You can use the built in installer and upgrader, or you can install the plugin manually.

1. You can either use the automatic plugin installer or your FTP program to upload it to your wp-content/plugins directory the top-level folder. Don't just upload all the php files and put them in `/wp-content/plugins/`.

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Go to Setting -> My Social Widgets and enable widgets you want to use.

4. Go to Appearance -> Widgets and add SW-Recent Posts, SW-Facebook, SW-Twitter widget.

5. Copy Short Code from setting page above and use on page/post content. 

6. That's it!
